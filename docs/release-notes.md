# Complete Offline Wikipedia AI - v1.0.0

Chat with AI that has complete Wikipedia knowledge - 100% free, 100% offline, 100% private.

## What's Included

- DeepSeek R1 - State-of-the-art open-source AI model
- Complete English Wikipedia - 6+ million articles offline
- 100% Private - Everything runs locally
- One-line install - Fully automated setup
- Free forever - No API keys or subscriptions

## Quick Start

```bash
git clone https://github.com/macromeer/offline-wikipedia-rag.git
cd offline-wikipedia-rag
./install.sh
```

## Requirements

- OS: Linux or macOS
- Disk: 120GB free space
- RAM: 8GB minimum (16GB recommended)
- Time: 2-8 hours for first-time setup

## Features

- Interactive chat mode
- Single question CLI mode
- Auto-model detection
- Clean, readable output
- Full Wikipedia coverage
- Completely offline operation

Built with DeepSeek R1, Ollama, Kiwix, and Wikipedia.
